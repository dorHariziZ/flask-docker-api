Welcome to the Crypto Price API

This API allows you to retrieve real-time cryptocurrency data using CoinGecko.

Available Endpoints:

1. Ping Server

GET http://127.0.0.1:5003/ping

2. Get Live Price of Bitcoin (default)

GET http://127.0.0.1:5003/simple/price

3. Get Live Price of Ethereum

GET http://127.0.0.1:5003/simple/price?ids=ethereum&vs_currencies=usd

4. Get List of Supported Coins

GET http://127.0.0.1:5003/coins/list

5. Get Market Data for Top 10 Coins

GET http://127.0.0.1:5003/coins/markets?vs_currency=usd

6. Get Coin Details (Bitcoin)

GET http://127.0.0.1:5003/coins/bitcoin

7. Get Historical Chart Data for Ethereum (Last 30 Days)

GET http://127.0.0.1:5003/coins/ethereum/market_chart?vs_currency=usd&days=30

8. Get Historical Price of Bitcoin on a Specific Date

GET http://127.0.0.1:5003/coins/bitcoin/history?date=01-01-2024

9. Get All Coin Categories

GET http://127.0.0.1:5003/coins/categories

10. Get Coin Tickers (Ethereum)

GET http://127.0.0.1:5003/coins/ethereum/tickers

Usage

Use the above endpoints in your browser or API clients like Postman or `curl`.
